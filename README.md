# Pacman Game (FIT VUT Version)

Pacman Game is a school project for the IJA subject at FIT VUT.

## Installation

The project was built using [Maven](https://maven.apache.org/).

To run the game, execute the following command:

```bash
java -jar pacman.jar
```
